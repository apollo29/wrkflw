# Architecture Decision Records

Hier werden Architekturentscheidungen als kurze ADRs dokumentiert (eine Datei pro
Entscheidung, fortlaufend nummeriert, z. B. `0001-ports-and-adapters.md`).

Die festgelegten Grundsatzentscheidungen stehen in `PLAN.md` ("Festgelegte
Entscheidungen") und werden in Phase 9 als ADRs ausformuliert.
